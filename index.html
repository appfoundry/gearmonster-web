What is it?

GearMonster is a pet project over at AppFoundry - a software development team based in Kontich, Belgium. It allows you to input your stompbox collection, group effect pedals (by pedalboard for example), add pictures, keep a list of wanted pedals, add notes, etc.

How to try it?

GearMonster for Android is currently in development, but we plan a Beta Release somewhere in the near future. You can signup here for early Beta access!
